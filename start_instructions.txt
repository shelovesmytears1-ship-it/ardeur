Чтобы запустить локальный сервер и открыть сайт (localhost), откройте терминал в этой папке (ardeur) и введите следующие команды:

cd ardeur

1. Если вы только что скачали проект или добавили новые зависимости, сначала установите их:
   npm install

2. Чтобы запустить сайт (сервер разработки):
   npm run dev

После этого в терминале появится ссылка (обычно http://localhost:5173), по которой будет доступен ваш сайт. Откройте её в браузере.
